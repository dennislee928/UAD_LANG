# docs/specs/

This directory contains **formal language specifications** for UAD.

## Contents

Language specifications will be organized as follows:

- `CORE_LANGUAGE_SPEC.md`: Core `.uad` language specification
- `MODEL_DSL_SPEC.md`: Model DSL (`.uadmodel`) specification
- `IR_SPEC.md`: Intermediate Representation specification
- `MUSICAL_DSL_SPEC.md`: Musical DSL syntax and semantics
- `STRING_THEORY_SPEC.md`: String Theory semantics specification
- `ENTANGLEMENT_SPEC.md`: Entanglement semantics specification

## Purpose

These specifications serve as:
- Reference documentation for language features
- Design documents for new features
- Basis for academic papers and whitepapers
